    • Research

    I'm generally interested in data-intensive systems under the context of new workloads, new infrastructures and new hardware. Some of my past research is about resource-efficient database systems and self-learning storage systems. Currently, I'm working on efficent and effective DBMS testing methods.

    CIAO: An Optimization Framework for Client-Assisted Data Loading
    Cong Ding, Dixin Tang, Xi Liang, Aaron J. Elmore, Sanjay Krishnan
    ICDE'21 Short Paper
    [extended version in arXiv] [video] [slides]

    CIAO is a tunable system to make clients cooperate with the server to enable efficient partial loading and data skipping for a given workload. We proposed an efficient algorithm which would select near-optimal predicate set to push down within given budget.
    CrocodileDB: Efficient Database Execution through Intelligent Deferment
    Zechao Shang, Xi Liang, Dixin Tang, Cong Ding, Aaron J. Elmore, Sanjay Krishnan, Michael J. Franklin
    CIDR'20

    CrocodileDB is a research prototype system that integrates new ways of specifying timing information, new query execution policies, new task schedulers, and new data loading schemes.
